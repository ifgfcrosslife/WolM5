How it works

Alur singkat biar jelas saat dipakai.

  1. Isi Supabase URL, anon key, dan Bridge ID.
  2. Register bridge supaya row wol_bridges ada.
  3. Tambah PC ke wol_devices manual atau nanti dari M5.
  4. Queue WOL untuk membuat row di wol_commands.
  5. M5 bridge yang akan mengeksekusi command itu di LAN.
